# Troubleshooting Log

Catat setiap error yang ditemui dan cara mengatasinya di sini.
Format: tanggal | komponen | deskripsi error | solusi

---

## Template

**Tanggal:** YYYY-MM-DD
**Komponen:** (Kafka / Spark / Airflow / MinIO / PostgreSQL / dll)
**Error:**
```
tempel pesan error di sini
```
**Penyebab:** jelaskan kenapa error ini terjadi
**Solusi:** langkah yang dilakukan untuk mengatasi
